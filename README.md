# Box Head Studio

Browser-based image distortion tool for a single editable Box Head Warp effect.

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Features

- Upload an image locally in the browser.
- Drag and resize one rectangular warp field.
- Adjust strength, horizontal stretch, vertical compression, edge softness, warp size, and mesh resolution.
- Toggle the warp box and mesh preview independently.
- Export a clean PNG at original size, 2x, or 4x without preview overlays.
