# MeshWarp

Browser-based image distortion tool for editable mesh-warp effects.

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Features

- Upload an image locally in the browser.
- Add, select, drag, and resize multiple warp fields.
- Switch mesh shapes between rectangle, ellipse, diamond, and Tic-Tac-Toe lattice.
- Drag the four inner Tic-Tac-Toe lattice intersections for advanced mesh control.
- Toggle live warp previews while dragging, with Fast, Balanced, and Sharp quality levels.
- Keep a warp as a box or switch to freeform corners for manual shape editing.
- Adjust strength, horizontal stretch, vertical compression, edge softness, warp size, and mesh resolution.
- Toggle the warp box and mesh preview independently.
- Export a clean PNG at original size, 2x, or 4x without preview overlays.
